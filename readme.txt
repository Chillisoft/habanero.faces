Habanero Faces v2.5.0, 9 September 2010

This package contains the source of Habanero Faces, a UI library for use with Habanero Core that supports Windows forms and Visual WebGUI (an ajax library) development. It includes the following:

Habanero.Faces.Base.dll - core functionality shared by Win and VWG
Habanero.Faces.Win.dll - the Windows Forms library
Habanero.Faces.VWG.dll - the Visual WebGUI library

To use the Visual WebGUI library you will need to download Visual WebGUI Professional Studio (VS 2008) .NET 3.5, 6.4 beta 1 from http://www.visualwebgui.com
